O **Greenbelt Store** é um aplicativo de e-commerce focado na venda de buquês, flores e pelúcias. O projeto foi desenvolvido em Flutter para atender ao desafio proposto pela agência digital fictícia "Digital Horizon", com o objetivo de criar um app para apoiar o comércio e serviços locais.

## 🎯 Objetivos Técnicos Atingidos

O desenvolvimento deste aplicativo permitiu a aplicação de diversas competências exigidas pelo curso:

- **Desenvolvimento Mobile com Dart:** Implementação de código respeitando as características da linguagem e aplicando as melhores práticas de Programação Orientada a Objetos (POO), incluindo o uso de classes e subclasses.

- **Persistência de Dados:** Estruturação segura de informações, garantindo o armazenamento interno/local para dados sensíveis e cache do app, além de persistência externa.

- **Integração RESTful:** Consumo de serviços web e múltiplas APIs, realizando requisições assíncronas de manipulação de dados utilizando os métodos `GET`, `POST`, `PUT` e `DELETE` com formato `JSON`.

- **Construção de Interfaces Dinâmicas:** Criação de leiautes com navegação fluida, passagem de parâmetros entre telas, tratamento de eventos e manipulação de listas na interface do usuário.

## 🚀 Como Executar

### Pré-requisitos
- Flutter SDK instalado
- Dart SDK instalado
- Emulador Android/iOS ou dispositivo físico conectado

### Instalação

1. **Clone o repositório**
```bash
git clone https://github.com/seu-usuario/greenbelt-flutter.git
cd greenbelt-flutter
```

2. **Instale as dependências**
```bash
flutter pub get
```

3. **Execute o projeto**
```bash
flutter run
```
